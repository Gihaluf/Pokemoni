**Par programmu:**  
Objektorientētā programma par Pokémoniem ļauj izveidot dažāda tipa Pokémonus, pārvaldīt to atribūtus, izsaukt to metodes un rīkot kaujas starp Pokémoniem. Programma ir veidota ar polimorfismu, mantošanu un abstrakciju.
